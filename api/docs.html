API Documentation

Integrate hosted authentication into your application with three endpoints and zero boilerplate.

1. The Redirect

Direct your users to the Ageinx hosted auth page. We handle the entire login and signup flow.

# URL Format
https://ageinx.vercel.app/auth/{your-slug}

2. Callback & Token

Once authenticated, Ageinx redirects the user back to your callback_url. The JWT is appended as a URL fragment.

// Example: https://yourapp.com/callback#token=eyJhbGci...
const token = new URLSearchParams(window.location.hash.substring(1)).get('token');

3. Verify Identity

Send the token from your backend to verify the user's data and ensure the session is valid.

GET https://ageinx.vercel.app/auth/userinfo
Authorization: Bearer <token>

Response Format

Successful verification returns a clean user identity object.

{
  "user_id": "uuid-string",
  "email": "user@example.com",
  "developer_id": "your-dev-id",
  "token_type": "user_access"
}
