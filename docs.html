Complete Example

The entire flow in one place. We give you a token after login, you verify the token on your backend.

1. The Login Button (HTML)

Send users to your hosted login page.

<a href="https://ageinx.vercel.app/auth/YOUR_SLUG">Login</a>

2. The Callback (Frontend JS)

Grab the token from the URL when they return.

const token = new URLSearchParams(window.location.hash.substring(1)).get('token');
if (token) {
    localStorage.setItem('token', token);
    window.history.replaceState(null, null, window.location.pathname); // clean URL
}

3. The Verification (Backend API)

Make sure the token is real before giving them data.

GET https://ageinx.vercel.app/auth/userinfo
Authorization: Bearer <token>

React / Next.js

A simple hook to handle the redirect.

import { useEffect } from 'react';
import { useRouter } from 'next/router';

export default function AuthCallback() {
    const router = useRouter();

    useEffect(() => {
        const token = new URLSearchParams(window.location.hash.substring(1)).get('token');
        if (token) {
            localStorage.setItem('ax_token', token);
            router.push('/dashboard'); 
        } else {
            router.push('/login');
        }
    }, []);

    return <div>Logging in...</div>;
}

Node.js / Express

Protect your routes using this middleware.

const axios = require('axios');

async function requireAuth(req, res, next) {
    const token = req.headers.authorization?.split(' ')[1];
    if (!token) return res.status(401).json({ error: "Denied" });

    try {
        const { data } = await axios.get('https://ageinx.vercel.app/auth/userinfo', {
            headers: { 'Authorization': `Bearer ${token}` }
        });
        req.user = data; 
        next();
    } catch (error) {
        return res.status(401).json({ error: "Invalid token" });
    }
}

Python / FastAPI

import httpx
from fastapi import FastAPI, Depends, HTTPException
from fastapi.security import HTTPBearer

app = FastAPI()
security = HTTPBearer()

async def verify_user(credentials = Depends(security)):
    async with httpx.AsyncClient() as client:
        response = await client.get(
            "https://ageinx.vercel.app/auth/userinfo",
            headers={"Authorization": f"Bearer {credentials.credentials}"}
        )
        if response.status_code != 200:
            raise HTTPException(status_code=401, detail="Invalid token")
        return response.json()

Security

  • No Plaintext Storage: Passwords are heavily hashed and salted.
  • Absolute Isolation: Only the user ID and Email are returned to your app.
  • Token Expiry: Tokens expire in 60 minutes.

Common Mistakes

  • ❌ Trusting frontend data: Backend verification is mandatory.
  • ❌ Not clearing the URL hash: Clean the URL so users don't copy/paste their active token.
  • ❌ Ignoring 401 errors: Tokens expire. Catch 401s and redirect to login.
