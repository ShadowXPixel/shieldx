Aegrix
One Platform. Five Features.
Zero infrastructure headaches.

Aegrix bundles everything your web app's backend needs into a single, cohesive API — authenticate users, block threats, store secrets, scan content, and monitor your app with one integration.

What is Aegrix

Aegrix is a hosted authentication platform with five built-in features. You have to redirect your traffic to us and We handle the rest. Every feature is designed to work effeciently alongside the others — after the users does the process you get a Token which you can use to call our api and get the User Data in JSON.

We built it because we were tired of wiring together five different third-party services on every new project. Aegrix is the tool we wished existed.

It's production-ready from day one — built with security as a hard constraint, not a feature to add later.

Quick start
Up and running in minutes.
// npm install @codemango/aegrix

import { Aegrix } from '@codemango/aegrix';

const sx = new Aegrix(process.env.AEGRIX_KEY);

// ── Auth: verify a user token
const { user } = await sx.auth.verify(req.headers.authorization);

// ── Guard: enforce a rate limit
await sx.guard.check({ ip: req.ip, limit: 100, window: '1m' });

// ── Vault: store an encrypted secret
await sx.vault.set('stripe_key', process.env.STRIPE_SECRET);

// ── Scan: check a user-submitted URL
const { safe } = await sx.scan.url(userLink);

// ── Insight: log a custom event
sx.insight.log({ event: 'user.signup', userId: user.id });

Why we started Aegrix
